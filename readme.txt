项目说明：

LiteSetup：核心代码

LiteSetupUICompressor：程序打包器（用于打包安装程序）(生成Compressor.exe)

LiteSetupUI：Install安装程序（提供给用户 的安装程序）（生成Install.exe）

简介：
简易安装程序制作软件

功能
1.支持写入超大文件
2.支持超过4G单个文件
3.支持创建桌面快捷方式


文件说明
Compressor.exe  打包器
Install.exe安装程序
data 打包器生成的数据文件
data.dat 打包器生成的数据文件

example文件夹，示例安装程序





Compressor.exe  打包器说明
功能：制作data 和data.dat文件 
选项：
1.默认程序根目录： 目标程序目录
2.快捷方式exe文件名：需要创建的快捷方式在打包目录中的相对路径+文件名.exe


Install.exe安装程序说明
这是一个通用的安装程序通过data文件和data.dat文件
来安装
